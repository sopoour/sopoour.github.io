Remote-controlled Pi Robot (aka Elliot)

I'm using a RPi 3B+ with an Explorer HAT. Elliot can be Remote-controlled via a remote and is unlocked via a RFID sensor. The initial set-up is based on STS Pilot by Mark Dammer with a Python based Flask application for the Backend and JS/HTML/CSS for the remote Frontend. I'm planning to add some distance and air pollution sensors as well as to use some object detection.

Blockchain-based Car Sharing Prototype

This is a prototype as part of my Master Thesis built together with my thesis partner. It simulates the unlocking of a car by having a Raspberry Pi representing the server of the car, an RFID sensor as the car lock, and the RFID tag as the keyless option to open the door. The transaction is verified in the Blockchain Network of Hyperledger Fabric. The prototype is written mainly in Python & Node.js

Prototyping: Edge Computing with Computer Vision for E-Scooter

As part of a research group we've developed Computer Vision Edge framework to ensure the safety of E-scooter. As part of this project we've built a prototype in Azure with a VM predicting the outcome "sidewalk" or "no sidewalk". Read in our paper more about it.
